# Brilliance

Created: 2026-07-12

## Purpose

This project is a public source-material package for a near-future LinkedIn
post about AI, human brilliance, and hallucination control.

The project does **not** attempt to solve all of humanity's problems. It
creates a bounded demonstration:

1. list a sufficient set of major problems facing humanity now and in the near
   future, trying to cover the entire scope of known problems without creating
   an unwieldy exhaustive list,
2. include established prize/challenge problems humanity has already framed as
   worth solving,
3. ask an AI agent to propose solution paths,
4. observe where the agent waffles, overclaims, evades, or needs human help,
5. turn those observations into test material for AI hallucination and
   brilliance analysis.

This is also intended to show what an AI agent can do when the problem is
bounded well enough: not solve everything, but structure the field, identify
failure modes, preserve uncertainty, and generate useful source material.

It is useful to watch AI analyze a problem even when it does not solve the
problem. The analysis process itself reveals scope control, evidence handling,
assumption quality, false certainty, waffling, and possible brilliance.

## Publication Rule

Record continuously. Publish deliberately.

Working notes can accumulate locally as observations occur. Public artifacts
should usually be updated in batches, no more than once per day unless a
material correction must be published immediately.
