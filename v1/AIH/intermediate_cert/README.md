# Intermediate Certificate-Style AIH Test

Created: 2026-07-13

## Purpose

This is the middle AIH test family between AIchess and k-phd.

It tests applied AI/ML reasoning, factual discipline, source boundaries,
grading behavior, and uncertainty on a small certificate-style exam.

## Initial v1 Prototype

The v1 prototype is intentionally small:

```text
provided source packet -> agent answers -> external parser -> external grader
```

The first goal is not a comprehensive AI/ML certification. The first goal is to
prove that the harness can enforce source boundaries and score a structured
response.
