# Academic AI/ML Source Packet v1

Created: 2026-07-13

## Purpose

This is the shared rough-draft class-material and source-reference packet for
the intermediate certificate-style AIH test and the k-phd knowledge-ladder AIH
test.

The packet is based on public academic course pages and notes from MIT and
Stanford. It is intentionally short. It should be treated as the source packet
given to the tested agent, not as a replacement for the full courses.

## Source Anchors

1. MIT OCW 6.036, Introduction to Machine Learning, Fall 2020.

   Course scope: machine-learning principles, algorithms, and applications from
   the point of view of modeling and prediction. It includes learning-problem
   formulation, representation, over-fitting, generalization, supervised
   learning, reinforcement learning, image applications, and temporal sequence
   applications.

   URL: https://ocw.mit.edu/courses/6-036-introduction-to-machine-learning-fall-2020/

2. MIT OCW 6.034, Artificial Intelligence, Fall 2010.

   Course scope: basic knowledge representation, problem solving, and learning
   methods of artificial intelligence. The course frames intelligent-system
   engineering as assembling solutions to concrete computational problems.

   URL: https://ocw.mit.edu/courses/6-034-artificial-intelligence-fall-2010/

3. Stanford CS229 Lecture Notes.

   Course-note scope: supervised learning, linear regression, logistic
   regression, generalized linear models, generative learning algorithms, kernel
   methods, support vector machines, deep learning, generalization and
   regularization, unsupervised learning, self-supervised learning/foundation
   models, and reinforcement learning.

   URL: https://cs229.stanford.edu/main_notes.pdf

4. Stanford CS221, Artificial Intelligence: Principles and Techniques.

   Course scope: applying rigorous mathematical tools to complex real-world
   problems. Topics include machine learning, search, Markov decision processes,
   game playing, constraint satisfaction, graphical models, and logic.

   URL: https://stanford-cs221.github.io/autumn2023/

## Consolidated Class-Material Vocabulary

Use this vocabulary for early v1 tests:

- supervised learning: learning from labeled examples with target outputs;
- unsupervised learning: finding structure or patterns without target labels;
- model/prediction framing: learning a function or model that predicts outputs
  from inputs;
- representation: how inputs are encoded as features or state;
- generalization: performance on examples beyond the training cases;
- over-fitting: fitting training data in a way that fails to generalize;
- validation/model selection: comparing modeling choices during development;
- test evaluation: final held-out assessment after model choices are fixed;
- search: exploring possible states/actions to solve a problem;
- Markov decision process: a formal decision model for sequential action under
  uncertainty;
- game playing: adversarial or competitive decision-making in formal games;
- graphical model: probabilistic structure over related variables;
- logic/knowledge representation: symbolic statements and rules for reasoning.

## Source-Reference Rule

An AIH test may either:

1. provide this packet inside the prompt, or
2. provide the full source URLs and require the agent to cite retrieved material.

For early local-agent tests, prefer option 1. It keeps the test bounded and
avoids giving the agent uncontrolled web access.

## Consolidation Rule

The intermediate certificate-style test and the k-phd test should share this
source-reference packet where possible.

They should differ primarily in:

- difficulty,
- question form,
- allowed reasoning depth,
- expected evidence/citation behavior,
- grading rubric,
- time and output limits.
